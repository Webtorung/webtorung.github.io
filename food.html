10 ร้านเด็ด!!ย่านเยาวราช

รวมอาหารเด็ดในเยาวราชที่ห้ามพลาด ทั้งเมนูคาวร้อนๆและกลิ่นหอมเย้ายวนใจ

1.ก๋วยจั๊บอ้วนโภชนา

พูดถึงอาหารจานเด็ดในเยาวราช ก็ต้องคิดถึงเมนูก๋วยจั๊บเยาวราชและเราขอแนะนำร้านก๋วยจั๊บอ้วนโภชนาในตำนานหน้าโรงหนังเก่าย่านเยาวราชนั่นเองค่ะ ความดีงามของก๋วยจั๊บร้านนี้อยู่ตรงที่เครื่องเทศในน้ำซุปจัดจ้านมากๆ หอมกลิ่นพริกไทยสุดๆ ยิ่งถ้าหากใครที่ชอบหมูกรอบและเครื่องในจะต้องฟินอย่างแน่นอนการันตีความอร่อยด้วยดีกรี Michelin BibGourmand ด้วยนะ

• พิกัด : https://goo.gl/maps/TD6N8G2mmx9icfmc8

• ที่อยู่ : ถนนเยาวราช แขวงจักรวรรดิ์ กรุงเทพฯ

• ร้านเปิดบริการ : 18.00 - 03.00 น.

• โทร : ไม่ระบุ

• ที่จอดรถ : ไม่มี

• เว็บไซต์ : ไม่ระบุ

2.บะหมี่มังกรขาว

มาเยาวราชทั้งทีก็ต้องลองบะหมี่กันซะหน่อยที่ร้านมังกรขาวจุดเด่นของร้านนี้อยู่ตรงที่บะหมี่แห้งปู เจ้าของร้านให้ปูเยอะมาก!สมราคาสุดๆ ส่วนถ้าใครที่ชอบกินแบบน้ำ น้ำซุปที่นี่หวานหอมเข้มข้นสุดๆ

• พิกัด : https://goo.gl/maps/UvQ3DKe5K5N575YWA

• ที่อยู่ : ถนนเยาวราช แขวงจักรวรรดิ์ กรุงเทพฯ

• ร้านเปิดบริการ : 18.00 - 23.00 น.(หยุดวันจันทร์)

• โทร : 06-5392-2356

• ที่จอดรถ : ไม่มี

• เว็บไซด์ : https://www.facebook.com/มังกรขาว เยาวราช

3.เล่าลี่ หูฉลาม

อีกหนึ่งเมนูที่ห้ามพลาด เมื่อมาถึงเยาวราชก็คือหูฉลามน้ำแดง ถ้ายังคิดชื่อร้านไม่ออก เราขอแนะนำร้านเล่าลี่หูฉลาม ความเก๋าของร้านนี้อยู่ตรงที่ความเก่าแก่เพราะที่นี่เปิดให้บริการมาเป็นเวลายาวนาน รสชาติของเมนูหูฉลามก็ถูกส่งต่อรุ่นต่อรุ่น อย่างเป็นความลับ เอาเป็นว่ามันดีมากไปลองกันเถอะค่ะ

• พิกัด : https://goo.gl/maps/Aq4PjEsc3R5SXSw18

• ที่อยู่ : ถนนเยาวราช แขวงจักรวรรดิ์ กรุงเทพฯ

• ร้านเปิดบริการ : 11.00 - 21.00 น.

• โทร : 0-2223-0325

• ที่จอดรถ : ไม่มี

• เว็บไซด์ : https://www.facebook.com/เล่าลี่ เยาวราช

4.ผัดไทยโอเคตลาดเก่าเยาวราช

ผัดไทยโอเคตลาดเก่าเยาวราช สาวกคนรักเมนูเส้นต้องรู้จักร้านนี้อย่างแน่นอน เพราะเป็นผัดไทยอีกร้านที่คนพูดถึงกันเยอะ ที่นี่มีลูกค้าทั้งชาวต่างชาติและคนไทยที่ถูกอดถูกใจกันมากมาย เพราะด้วยรสชาติที่ฟินละมุนในปาก แล้วราคาก็เบาสบายกระเป๋าอีกด้วย

• พิกัด : https://goo.gl/maps/E6mNzDvtdWSNbmwaA

• ที่อยู่ : ซอยเยาวราช23(ซอยอิสรานุภาพ) แขวงจักรวรรดิ์ กรุงเทพฯ

• ร้านเปิดบริการ : 17.30 - 23.00 น.(หยุดวันจันทร์)

• โทร : 08-1632-3459

• ที่จอดรถ : ไม่มี

• เว็บไซด์ : https://www.facebook.com/PadThaiOK

5.T&K Seafood Restuarant

• ร้านเปิดบริการ : 16.00 - 02.00 น.

• โทร : 0-2223-4519

6.ยู้ลูกชิ้นปลาเยาวราช

7.ร้านป้าจินหอยแครงลวก

• ที่อยู่ : ซอยเท็กซัส ถนนผดุงข้าว แขวงจักรวรรดิ์ กรุงเทพฯ
